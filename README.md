# TUeats
HOW TO RUN [vs code]:
Clone repo, open TUEats.html, right click and 'open in other browser', or 'open with live server'[extension] 
 

Simple integration of HTML,CSS,JS, focusing on form input validation, and passing data to another html file.

used embedded <script> within TUEats.html, created functions to validate form input [password, username, email ...]

used a GET request to tueats_completed.html, and passed the form data using localStorage
